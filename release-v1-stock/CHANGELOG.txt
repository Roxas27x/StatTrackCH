StatTrack - Changelog

2026-04-23
- Added per-song, per-section export template overrides so individual sections can use their own custom text formatting.
- Added section counter seeding in the Export Templates editor for Attempts, FCs Past, and Killed the Run.
- Preserved saved section counter values when their exports are disabled instead of showing them as reset to 0.
- Added a direct folder shortcut in the Export Templates panel so players can open the exact current section export folder they are editing.
- Added NoteSplit attempts editing by double-clicking the attempts count in the desktop NoteSplit header.

2026-04-22
- Added a built-in Export Templates editor so text exports can be customized without waiting for a new build.
- Added live animated main menu controls for tint color, wisp color, wisp size, and layer strengths.
- Added automatic and manual NoteSplit section-list scrolling for long songs so the active section stays visible.
- Improved export template editor focus and input blocking so typing in the editor does not bleed through to the main menu or overlay.
- Polished the animated menu slider controls and related overlay UI for clearer visuals and easier tuning.

2026-04-20
- Reduced gameplay hitching by idling export work when unused and moving more save/export work off the gameplay path.
- Improved desktop overlay and NoteSplit rendering so the overlay does less work during songs.
- Fixed NoteSplit toggles reverting when enabled from the menu or during a song.
- Fixed NoteSplit misses being credited to the wrong section at split boundaries.
- Fixed NoteSplit large-font clipping in the header, footer, attempts block, and section rows.
- Updated NoteSplit row backgrounds so they scale cleanly with larger row heights.

2026-04-19
- Added a previous valid run column to NoteSplit.
- Previous valid run now preserves only the sections you actually reached before a restart.
- Updated NoteSplit colors so previous-run and PB misses stay red.
- Updated NoteSplit colors so current/previous 0s are gold while PB 0 remains white.

2026-04-18
- Added NoteSplit Mode for LiveSplit-style per-section miss tracking.
- Added saved per-song, per-difficulty, and per-song-speed NoteSplit personal bests and attempts.
- Added a separate NoteSplit desktop window with taskbar presence for OBS capture.
- Added drag-to-move, drag-to-resize, font selection, and persistent NoteSplit window styling.
- Fixed section miss attribution so misses are credited to the correct chart section at boundaries.
- Hid NoteSplit automatically during bot play and practice mode.
- Disabled the stock auto-pause behavior that triggers when a controller disconnects mid-song.

2026-04-15
- Added a player-facing release pack with install.ps1 and uninstall.ps1.
- Added automatic backup/restore of Assembly-CSharp.dll during install and uninstall.
- Added a ready-to-zip release output under dist-v1-stock\release.
- Slimmed and cleaned the stock-v1 release surface for public distribution.
- Simplified section OBS export into a single Section Summary checkbox and multiline summary.txt output.
- Kept OBS exports opt-in by default to reduce unnecessary hitching.
