StatTrack v1.0.4

This release focuses on smoother gameplay and NoteSplit polish.

- Reduced hitching by idling export work when it is not needed and moving more save/export work off the gameplay path.
- Improved desktop overlay and NoteSplit rendering so the overlay does less work during songs.
- Fixed NoteSplit misses being credited to the wrong section at split boundaries.
- Fixed NoteSplit mode toggles reverting when enabled from the menu or during a song.
- Fixed NoteSplit large-font clipping in the header, footer, attempts block, and section rows.
- Updated NoteSplit row backgrounds so they scale cleanly with larger row heights.

For the safest install, use StatTrack on a clean Clone Hero copy instead of installing over an already modded folder.
