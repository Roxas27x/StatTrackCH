StatTrack v1.0.7

This release focuses on smoother gameplay, lighter live exports, and a cleaner installer baseline.

- Reduced in-song work so the tracker only reads, tracks, exports, or repaints what active settings actually need.
- Optimized NoteSplit and desktop overlay updates to avoid unnecessary redraws while a song is running.
- Restored NoteSplit attempt updates without forcing unrelated OBS export writes.
- Simplified section FC exports so only checked current-song sections stay in the live export path.
- Added clean-baseline installation: the installer renames existing game files, installs bundled clean Assembly-CSharp.dll and sharedassets1.assets files, then applies StatTrack on top.
- Replaced the stock main-menu News feed with StatTrack update history links.
- Added a StatTrack Discord card to the top of the News feed.
- Added Tab, Right Arrow, and Left Arrow keyboard focus controls for the News feed.
- Removed stale Ctrl+O help text; the overlay hotkeys are Home and F8.

Installer note: existing Assembly-CSharp.dll and sharedassets1.assets files are renamed, not deleted.
