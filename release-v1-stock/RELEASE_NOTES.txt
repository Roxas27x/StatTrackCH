StatTrack v1.0.3

Fixed the GitHub update checker on the main menu. Older StatTrack builds should now correctly detect when a newer release is available.

Tracked section exports now include raw value-only counter files for OBS:
- attempts.txt
- fcs_past.txt
- killed_the_run.txt

Tracked section folders no longer export:
- start_time.txt
- tracked.txt

summary.txt is still included and remains the formatted multi-line section summary.

For the safest install, use StatTrack on a clean Clone Hero copy instead of installing over an already modded folder.
