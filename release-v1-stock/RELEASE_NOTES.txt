Clone Hero Section Tracker - Release Notes

- OBS exports are opt-in and configurable from the in-game overlay editor.
- Desktop overlay support is included for borderless/windowed play.
- Completed runs are stored in memory.json and exported to OBS only when enabled.
- Section exports are streamlined into a single multiline Section Summary output for OBS convenience.
- Main menu support is included so export defaults can be chosen before entering a song.
- Install and uninstall scripts are included for patching a vanilla Clone Hero v1 folder safely.
