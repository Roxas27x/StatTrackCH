Clone Hero Section Tracker - Release Notes

- NoteSplit Mode adds a LiveSplit-inspired section tracker based on per-section note misses.
- NoteSplit runs in its own desktop window with taskbar visibility for easier OBS capture.
- NoteSplit supports saved personal-best misses by section, previous-section summaries, and total song attempts.
- Practice mode and bot play now hide NoteSplit entirely so no progress is tracked there.
- Unplugging an input device during a song no longer forces the game to auto-pause.
- OBS exports remain opt-in and configurable from the in-game overlay editor.
- Desktop overlay support is included for borderless/windowed play.
- Install and uninstall scripts are included for patching a vanilla Clone Hero v1 folder safely.
