StatTrack v1.0.2

Added a new previous valid run column to NoteSplit.
Previous valid run only comes from non-practice, non-bot runs.
Restarting mid-run now preserves the section results you actually reached, while unreached sections stay --.
Current run, previous run, and PB are now easier to compare side by side as a result.

Color updates

Previous-run misses now stay red instead of white.
PB misses now stay red instead of white.
Any 0 in the current run or previous run is now gold.
PB 0 stays white so it remains visually distinct.
